# API Documentation

This documentation site covers all of the Uploadjoy APIs and packages you can integrate into your apps. {{ className: 'lead' }}

## Getting started

To get started, create a new project within Uploadjoy. An API key is automatically created, simply go to your project's developer settings and copy it. Once you have created and saved your API key, you are ready to start building with Uploadjoy. {{ className: 'lead' }}

<Note>API keys are scoped to the specific project it was created in.</Note>
